# Некоторые задачи и проекты из курса ""Искусство разработки на современном C++

### 1. [Simple_database](https://github.com/TimSh24/CPP_Belts/tree/main/Simple_database)
Реализация простой базы данных      
**Навыки:** ООП, перегрузка операторов, обработка исключений  

### 2. [Search_Server](https://github.com/TimSh24/CPP_Belts/tree/main/Search_Server)  
Реализация поисковой системы      
**Навыки:** многопоточность  

### 3. [Deque_template](Deque_template.cpp)
Реализация шаблона класса Deque на осное двух векторов   
**Навыки:** шаблоны классов, константные и неконстантные методы

### 4. [Hotel_booking](Hotel_booking.cpp)
Система бронирования отелей        
**Навыки:** ООП, алгоритмы

### 5. [Josephus_permutation](Josephus_permutation.cpp)
Считалка Иосифа    
**Навыки:** move_семантика

### 6. [Simple_vector_template](Simple_vector.h)
Шаблон класса SimpleVector    
**Навыки:** шаблоны классов, move-семантика

### 7. [Explore_key_Words](ExploreKeyWords.cpp)
Реализация функции Stats ExploreKeyWords(const set<string>& key_words, istream& input), выполняющей подсчет слов из key_words во входном потоке       
**Навыки:** многопоточность  

### 8. [Concurrent_map](Concurrent_map.cpp)
Реализация хранилища ConcurrentMap, потдерживающего параллельный доступ к элементам       
**Навыки:** многопоточность  

### 9. [Secondary_index](Secondary_index.cpp)
Реализовано хранилище со вспомогательной структурой, называемой "вторичный индекс", для быстрого поиска.      
**Навыки:** эффективное использование ассоциативных контейнеров

### 10. [Figure_set](Figure_set.cpp)
Реализованы базовый класс Figure, классы-наследники Circle, Triangle, Rect, переопределяющие его виртуальные методы,  
функция CreateFigure, которая в зависимости от содержимого входного потока создаёт shared_ptr<Rect>, shared_ptr<Triangle> или shared_ptr<Circle>.      
**Навыки:** полиморфизм

### 11. [Cache](Cache.cpp)
Реализован LRU cache для электронной библиотеки      
**Навыки:** shared_ptr, многопоточность
